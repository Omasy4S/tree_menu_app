Наши услуги

Эта страница демонстрирует работу с вложенными пунктами меню. Если в меню есть подпункты под "Услуги", они будут раскрыты.

Примеры услуг

В админ-панели можно создать подпункты меню для этой страницы, например:

  • Веб-разработка
  • Мобильные приложения
  • Консалтинг
  • Поддержка

Каждый подпункт может иметь свой URL, например: /services/web-development/

Создание вложенного меню

Шаг 1: В админке создайте пункт меню "Услуги" с URL /services/
Шаг 2: Создайте дочерние пункты, указав в поле "Родительский пункт" пункт "Услуги"
Шаг 3: Укажите URL для каждого дочернего пункта
Шаг 4: Используйте поле "Порядок сортировки" для упорядочивания пунктов

→ Пример вложенной страницы (Веб-разработка)
